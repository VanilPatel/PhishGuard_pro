# PhishGuard: Simple Email Security Scanner

A Python-based tool to demonstrate how phishing indicators are detected in text.

## 🔍 How it works
The script analyzes text files for:
- **Social Engineering:** Scans for keywords like "urgent" or "suspended".
- **Malicious Links:** Detects raw IP address URLs.
- **Urgency Scoring:** Calculates a risk level based on the findings.

## 🛠️ Usage
1. Place an email text file in the `data/` folder.
2. Run the script from the root directory:
   ```bash
   python src/main.py
   ---

### How to Run it in VS Code
1. Open your terminal in VS Code (**Ctrl + `**).
2. Make sure you are in the main `Phishing-Detector` folder.
3. Type: `python src/main.py`
4. You should see the scan results immediately!
